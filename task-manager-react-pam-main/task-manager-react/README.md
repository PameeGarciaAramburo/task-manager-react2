# Task Manager

Aplicación web para la gestión de tareas desarrollada con React, Express, PostgreSQL y Prisma.

<!-- BADGE_CI -->

## Instalación local 🚀

```bash
git clone https://github.com/PameGarciaAramburo/task-manager-react2.git
cd task-manager-react-pam-main
npm install
```

### Variables de entorno

Crea un archivo `.env` con las siguientes variables:

```env
DATABASE_URL=
JWT_SECRET=
PORT=
```

## Comandos disponibles 📜

| Comando | Descripción |
|---------|-------------|
| `npm run dev` | Ejecuta el proyecto en modo desarrollo |
| `npm run build` | Genera la versión de producción |
| `npm test` | Pendiente (Sesión 3) |

## Base de datos 🗄️

El proyecto utiliza PostgreSQL con Prisma para gestionar la base de datos.# prueba de protección
